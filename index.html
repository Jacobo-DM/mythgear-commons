MythGear Commons

Open, human-scale tools for movement, rest, warmth, and repair.

README

What this is

MythGear is a commons of practical, body-centered tools. Each offering is designed to be repairable, adaptable, and rooted in lived human experience— not optimization, surveillance, or extraction.

These are not products. They are companions.

Core offerings

DreamHearth

A place where nothing is asked of you but rest. A modular nest-and-shell bedding system that grows with a human across a lifetime.

SpiderVine

Body-powered line travel and movement gear. Learning, play, and utility through direct engagement with gravity.

SunHearth

Passive warmth, safety, and environmental comfort. Tools for creating hearth without flame.

Gloves

Tactile sensing gear for hands. Attention, feedback, and gentle accountability through touch.

Shoes

Ground-contact footwear experiments. Balance, listening, and movement awareness.

Cards

Prompt and reflection tools. Small artifacts that shift attention rather than command it.

Code Tape

Pattern language and documentation experiments. How knowledge travels when it stays repairable.

Commons ethic

MythGear favors:

  • repair over replacement
  • learning over automation
  • dignity over control
  • presence over performance

If a tool removes agency, it does not belong here.

MythGear Commons — part of the Fox’s Anvil ecosystem. Open tools for inhabiting a human life.
